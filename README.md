# Website Starter

Clean starter project for building a website from scratch without recreating the initial Vite, React, Tailwind, Router, and shadcn setup.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- React Router
- shadcn/ui

## Development

```sh
npm install
npm run dev
```

## Available scripts

- `npm run dev` starts the development server
- `npm run build` creates a production build
- `npm run lint` runs ESLint
- `npm run preview` previews the production build locally

## Starter structure

- `src/App.tsx` defines the router
- `src/pages/Index.tsx` is the starter home page
- `src/pages/NotFound.tsx` is the default fallback route
- `src/components/ui/*` contains reusable shadcn/ui primitives
